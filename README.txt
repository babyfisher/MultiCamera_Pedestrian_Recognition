
Code for paper "Pedestrian Recognition in Multi-Camera Networks Based on Deep Transfer Learning and Feature Visualization"

1. 
Download features and auxiliary data of WARD datset from 
https://drive.google.com/open?id=1cIgZPMA9BvR2ExRyrlpXWaj4Mfdn24xE

https://drive.google.com/open?id=1EjbArXRk4a-_fohK7TWCRswSo_n3ANtT

2. Extract ward and prid folders at here

3. Run test_ward.m to test WARD dataset

4. Run test_prid.m to test PRID dataset
